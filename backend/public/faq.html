FAQ

Does this demo use JavaScript?

No—this page is built with HTML and CSS only. Hook it up to your backend when you're ready.

How are duplicates prevented?

On the server, compare content hashes (e.g., SHA‑256) before persisting. The UI simply reflects the result.

Can I enable automatic logout?

Yes. Configure session timeouts on the server and surface countdown & re‑auth prompts in the client.
